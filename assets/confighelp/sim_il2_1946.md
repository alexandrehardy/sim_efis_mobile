# IL-2 1946 configuration

1. Add the devicelink configuration to `conf.ini` in the IL-2 1946 installation directory:

        [DeviceLink]
        port=11946

2. Setup Sim-EFIS so that the host setting points to the machine running IL-2, and specify port 11946.
3. Run IL-2 1946.
4. Sim-EFIS should be able to establish contact as soon as you have entered a mission.
