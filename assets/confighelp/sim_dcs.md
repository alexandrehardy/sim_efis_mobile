# DCS configuration

1. Please ensure that the network interface to listen on is selected.
2. Download [export.lua](export.lua) by clicking on the blue text for the file name.
3. Place the `export.lua` in `$HOME\Saved Games\DCS\Scripts\Export.lua`. For example `C:\Users\YourName\Saved Games\DCS\Scripts\Export.lua`.
4. Take note of the IP address of the machine running DCS.
5. Configure Sim-EFIS to scan the network that DCS is running on, or enter the IP of the host to connect to.
6. The default connection port is 45000, you only need to change that if you modify the `Export.lua` file.
